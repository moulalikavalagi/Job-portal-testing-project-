# Job Portal Web Application - Testing Project

## About This Project
A complete manual testing project for a Job Portal web application.
This project covers end to end testing including test planning, 
test case design, bug reporting and test summary reporting.

## Project Structure
| File | Description |
|---|---|
| TEST_PLAN.md | Complete test plan with scope and objectives |
| TC_Registration_Login.md | 15 test cases for registration and login |
| TC_Job_Search.md | 15 test cases for job search and filters |
| TC_Job_Application.md | 15 test cases for job application process |
| BUG_REPORTS.md | 5 detailed bug reports with reproduction steps |
| TEST_SUMMARY_REPORT.md | Final test summary with metrics and recommendations |

## Skills Demonstrated
