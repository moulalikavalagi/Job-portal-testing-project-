# Test Plan: Job Portal Web Application

## 1. Objective
To verify that all features of the job portal work correctly for three users: Job Seeker, Employer, and Admin.

## 2. Scope
### In Scope:
- User Registration and Login
- Job Seeker Profile Creation
- Job Search and Filters
- Job Application Process
- Employer Job Posting
- Admin Dashboard

### Out of Scope:
- Performance Testing
- Security Testing

## 3. Test Types
- Functional Testing
- UI Testing
- Boundary Value Testing
- Negative Testing
- API Testing

## 4. Tools Used
- Test Cases: Excel/Google Sheets
- Bug Reporting: JIRA format
- API Testing: Postman
- Test Management: GitHub

## 5. Entry Criteria
- Requirements are clear
- Test environment is ready

## 6. Exit Criteria
- All test cases executed
- No critical bugs open
